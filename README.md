# 💰 思源笔记贵金属监控插件

自动爬取工商银行贵金属价格数据，支持多品种监控、实时价格预警、历史数据可视化等功能。

## ✨ 功能特性

### 📊 多品种支持
支持监控以下9种贵金属品种：
- 人民币账户黄金、白银、铂金、钯金
- 美元账户黄金、白银、铂金、钯金
- 代理实物贵金属

每个品种独立存储历史数据，可单独启用/禁用监控。

### 🔔 智能价格预警
支持四种预警类型，触发时通过思源通知提醒：

| 预警类型 | 说明 | 示例 |
|---------|------|------|
| **价格上涨至** | 价格达到设定值时提醒（避免重复提醒） | 黄金涨至 1000 元/克时提醒 |
| **价格下跌至** | 价格低于设定值时提醒（避免重复提醒） | 黄金跌至 800 元/克时提醒 |
| **日涨跌幅超%** | 当天涨跌超过设定百分比时提醒 | 当日涨跌超过 3% 时提醒 |
| **价格变化超%** | 相比上次查询变化超过设定百分比时提醒 | 单次波动超过 2% 时提醒 |

### 📈 实时价格显示
- 顶栏按钮支持显示实时价格徽章
- 可选择显示任意品种的当前价格
- 价格涨跌使用不同颜色标识

### 📉 历史数据可视化
- 点击顶栏 📈 按钮打开数据面板
- 实时统计：当前价格、最高价、最低价、涨跌幅
- 可视化价格走势图（支持 1天/7天/30天/全部 时间范围）
- 详细历史数据表格，每个品种最多保留最近 2000 条记录

### ⏰ 自动查询
- 支持多种刷新频率：10秒、30秒、1分钟、5分钟、10分钟、15分钟、30分钟
- 支持手动刷新数据
- 每个品种的历史数据独立存储

## 🚀 使用指南

### 第一步：启用监控品种

1. 点击思源笔记顶栏的 📈 按钮打开面板
2. 在左侧品种列表中勾选要监控的品种
3. 每个品种可以独立设置预警规则

### 第二步：设置刷新频率

1. 点击面板右上角的"设置"按钮
2. 选择"查询间隔"（默认5分钟）
3. 可选项：10秒、30秒、1分钟、5分钟、10分钟、15分钟、30分钟

### 第三步：配置顶栏价格显示

1. 在设置面板中选择"顶栏显示价格"
2. 选择要在顶栏按钮上显示价格的品种
3. 选择"不显示"则隐藏价格徽章

### 第四步：设置预警规则

1. 在左侧品种列表中点击要设置的品种
2. 在"预警概览"区域点击"添加预警"或"编辑预警"
3. 设置预警条件（留空则不启用该预警）
4. 支持设置多个价格阈值（用逗号分隔）

### 第五步：查看数据

1. 点击左侧品种列表选择要查看的品种
2. 查看实时价格、走势图和历史数据
3. 使用时间范围按钮切换图表显示范围

## ⚙️ 详细配置说明

### 全局设置

| 设置项 | 说明 | 默认值 |
|-------|------|--------|
| 启用自动查询 | 是否开启定时自动查询 | 开启 |
| 查询间隔 | 自动查询的时间间隔 | 5分钟 |
| 查询后通知 | 每次查询后发送价格通知 | 关闭 |
| 顶栏显示价格 | 在顶栏按钮显示指定品种价格 | 人民币账户黄金 |

### 品种设置

每个品种可以独立配置：

| 设置项 | 说明 | 默认值 |
|-------|------|--------|
| 启用监控 | 是否监控该品种 | 仅人民币账户黄金开启 |
| 价格上涨至 | 价格≥设定值时提醒 | 空 |
| 价格下跌至 | 价格≤设定值时提醒 | 空 |
| 日涨跌幅超% | 当日涨跌超过设定百分比时提醒 | 3% |
| 价格变化超% | 相比上次变化超过设定百分比时提醒 | 2% |

## 🌐 数据来源

本插件数据来源于 [工商银行贵金属页面](https://m.icbc.com.cn/mpage/precious-metal/list)，使用 WebView 加载动态页面并解析实时价格数据。

由于工商银行页面使用 JavaScript 动态加载数据，插件会创建一个隐藏的 WebView 来渲染页面，等待数据加载完成后提取价格信息。

> ⚠️ **注意**：本页面贵金属行情报价仅供参考，具体请以实际交易时的报价为准。

## 🔒 隐私说明

- 所有历史数据保存在本地，不会上传到任何服务器
- 仅向工商银行官网发送请求获取价格数据
- 预警通知仅在本地通过思源笔记发送

## ❓ 常见问题

### Q: 为什么收不到预警通知？
A: 请检查：
1. 品种是否已启用监控
2. 预警规则是否已设置
3. 思源笔记的通知权限是否开启
4. 自动查询是否已开启

### Q: 为什么显示"暂无数据"？
A: 新启用的品种需要等待下次自动查询或手动点击"立即查询"才能获取数据。

### Q: 如何清空历史数据？
A: 在面板左侧品种列表中，点击对应品种旁的"🗑️"按钮，或在设置中点击"清空所有数据"。

### Q: 可以同时监控多个品种吗？
A: 可以。勾选多个品种即可同时监控，每个品种有独立的预警规则和历史数据。

### Q: 预警会重复提醒吗？
A: 不会。价格上涨/下跌预警会在条件首次满足时提醒，之后价格回到阈值以下/以上后再次满足条件才会再次提醒。日涨跌幅预警每天只提醒一次。

### Q: 顶栏价格不显示怎么办？
A: 请检查：
1. 设置中的"顶栏显示价格"是否选择了品种
2. 该品种是否已获取到数据（等待自动查询或手动刷新）

### Q: 为什么查询有时会超时？
A: 由于使用 WebView 加载工商银行页面，需要等待页面完全渲染后才能获取数据。如果网络较慢或页面加载失败，可能会导致超时。建议：
1. 检查网络连接
2. 增加查询间隔时间
3. 手动点击"立即查询"重试

### Q: WebView 会影响性能吗？
A: WebView 是隐藏加载的，且每次查询后会立即销毁，对性能影响很小。查询间隔建议设置为 30 秒以上，避免频繁加载页面。
